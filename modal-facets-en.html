You can use filters to make search results more accurate.

If you use multiple filters with checkboxes  within one section (e.g. "Loan type") OR logical operator is applied. 

In all other cases AND logical operator is applied.

Click the arrow symbol to expand hidden lists of additional filters.

You can remove applied filters one by one (the pink cross pictogram ) or all at once ("Remove filters" ).

 

When using filter "Institution" and "Year of publication" confirm your choice by "Use filter" button.

The use of icons in filter "Institution":

  • - save a selection for later use. This icon is displayed only to the logged-in user.

  •   - load your saved selection. This icon is displayed only to the logged-in user.

  •  - select libraries where you are registered. This icon is displayed only to the logged-in user.

  •  - find the nearest libraries.

If you are using any filter and you want to enter a new search query, you can choose the option to "Search with current filters" (checkbox under the search field). This option is set as default and can be unchecked.
